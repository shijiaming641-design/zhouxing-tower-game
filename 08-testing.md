## 6. 测试自动化

### 6.1 单元测试示例 (NUnit)

```csharp
[TestFixture]
public class DamageCalculatorTests {
    
    [Test]
    public void TestWinDamage() {
        int damage = CalculateDamage(10, 2, RPSChoice.Rock, 
                                    RPSChoice.Scissors, 0);
        Assert.AreEqual(12, damage, 1);
    }
    
    [Test]
    public void Test3StreakBonus() {
        int damage = CalculateDamage(10, 2, RPSChoice.Rock, 
                                    RPSChoice.Scissors, 3);
        Assert.AreEqual(14, damage, 2);
    }
    
    [Test]
    public void TestMinDamage() {
        int damage = CalculateDamage(5, 10, RPSChoice.Paper, 
                                    RPSChoice.Rock, 0);
        Assert.GreaterOrEqual(damage, 1);
    }
}
```

### 6.2 集成测试

public class BattleIntegrationTests {
