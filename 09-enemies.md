# 敌人图鉴

## 1. 普通敌人（1-4层）

### 1.1 蒸汽劳工

| 属性 | 数值 |
|------|------|
| 生命值 | 30 |
| 攻击力 | 8 |
| 防御力 | 2 |
| 出拳倾向 | 33% / 33% / 34% |

**背景**：最基础的蒸汽工人，负责维护塔内的管线运行。他们没有特殊能力，只是单纯地进行日常劳作。

**战斗特点**：
- 完全随机出拳，无规律可循
- 适合作为玩家的教学对手

**击败奖励**：10-15金币

---

### 1.2 齿轮守卫

| 属性 | 数值 |
|------|------|
| 生命值 | 40 |
| 攻击力 | 10 |
| 防御力 | 4 |
| 出拳倾向 | 石头60% / 剪刀20% / 布20% |

**背景**：齿轮会的基层守卫，身披齿轮铠甲，坚守"石头"即"坚硬不可摧"的信念。攻击节奏稳定。

**战斗特点**：
- 60%概率出石头，偏好防御
- 每3回合进入"防御姿态"，获得3点护盾

**技能**：
- **防御姿态**：每3回合，获得3点护盾

**击败奖励**：12-18金币

---

### 1.3 管线鼠

| 属性 | 数值 |
|------|------|
| 生命值 | 25 |
| 攻击力 | 12 |
| 防御力 | 0 |
| 出拳倾向 | 石头30% / 剪刀50% / 布20% |

**背景**：栖息在蒸汽管线中的变异生物，动作敏捷，喜欢快速出招。

**战斗特点**：
- 高攻击低血量，典型的刺客型
- 50%概率出剪刀，偏好进攻
- 闪避率15%（攻击有15%概率无效）

**技能**：
- **闪电突袭**：有15%概率闪避玩家攻击

**击败奖励**：10-15金币

### 1.4 蒸汽学徒

| 属性 | 数值 |
|------|------|
| 生命值 | 35 |
| 攻击力 | 11 |
| 防御力 | 2 |
| 出拳倾向 | 石头20% / 剪刀40% / 布40% |

## 2. 精英敌人（5-9层）

### 2.1 蒸汽监工

| 属性 | 数值 |
|------|------|
| 生命值 | 80 |
| 攻击力 | 15 |
| 防御力 | 5 |
| 出拳倾向 | 石头20% / 剪刀30% / 布50% |

**背景**：负责监督普通劳工的监工，性格暴躁，攻击附带"易伤"效果。

**战斗特点**：

- 50%概率出布，压制石头

**技能**：
- **威压**：攻击命中后，玩家获得1层"脆弱"（受到伤害+50%）

**击败奖励**：30-40金币，稀有卡牌

---

### 2.2 齿轮骑士

| 属性 | 数值 |
|------|------|
| 生命值 | 100 |
| 攻击力 | 18 |
| 防御力 | 8 |
| 出拳倾向 | 石头70% / 剪刀15% / 布15% |

**背景**：齿轮会的中坚力量，身经百战的骑士。坚信防御就是最好的进攻。

**战斗特点**：
- 70%概率出石头，极其保守

- **反击**：被克制时，有30%概率反击（反弹30%伤害）

### 2.3 锅炉精灵

| 属性 | 数值 |
|------|------|
| 生命值 | 70 |
| 攻击力 | 20 |
| 防御力 | 2 |
| 出拳倾向 | 33% / 33% / 34% |

**背景**：由蒸汽凝聚而成的元素生物，性格不稳定，容易暴怒。

**击败奖励**：30-40金币，稀有卡牌

## 5. 敌人设计原则

### 5.1 难度梯度

| 层数 | 难度 | 设计重点 |
|------|------|----------|
| 1-2 | 简单 | 教学，敌人单一技能 |
| 3-4 | 中等 | 引入组合技能 |
| 5 | 简单Boss | 单阶段，固定模式 |
| 6-7 | 困难 | 需要策略卡牌 |
| 8-9 | 极难 | 多技能组合 |
| 10 | 终极Boss | 多阶段，策略深度 |

### 5.2 出拳倾向设计

| 敌人类型 | 倾向风格 | 策略对应 |
|----------|----------|----------|
| 防御型 | 石头60%+ | 玩家应出布 |
| 攻击型 | 剪刀/布50%+ | 玩家应出石头 |
| 均衡型 | 33%/33%/34% | 需观察后决策 |
| 随机型 | 完全随机 | 运气成分 |

### 5.3 技能设计原则

- 每个敌人至少1个核心技能
- 精英敌人2-3个技能
- 技能效果尽量直观易理解
- 避免过于复杂的连锁效果
