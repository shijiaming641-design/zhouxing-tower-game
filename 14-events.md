# 每周挑战与赛季系统

## 1. 赛季概述

### 1.1 赛季结构

| 项目 | 内容 |
|------|------|
| 赛季周期 | 4周（28天） |
| 赛季主题 | 每周轮换特殊规则 |
| 奖励 | 限定卡牌/头像/金币 |
| 解锁 | 达到第5层后开放 |

### 1.2 赛季类型

| 赛季 | 主题 | 特殊规则 |
|------|------|----------|
| S1 | 齿轮之怒 | 敌人攻击+30% |
| S2 | 三角轮回 | 只能使用3种卡牌 |
| S3 | 极速挑战 | 回合数上限15 |
| S4 | 无限火力 | 技能无冷却 |

---

## 2. 每周特殊事件

### 2.1 第1周：齿轮之怒

**规则**：
- 所有敌人攻击力+30%
- 玩家初始生命+20

**设计意图**：
- 鼓励玩家采取防御策略
- 考验玩家生存能力

**限定奖励**：
| 达成条件 | 奖励 |
|----------|------|
| 通关 | 限定卡牌"钢铁意志" |
| 达到第8层 | 头像"齿轮战士" |
| 达到第5层 | 200金币 |

---

### 2.2 第2周：三角轮回

**规则**：
- 只能使用3种基础卡牌（石头/剪刀/布）
- 无法获得/购买其他卡牌
- 初始卡组变为3张石头+3张剪刀+3张布

**设计意图**：
- 回归纯粹猜拳
- 考验对基础机制的理解

**限定奖励**：
| 达成条件 | 奖励 |
|----------|------|
| 通关 | 限定卡牌"轮回之眼" |
| 达到第8层 | 头像"纯粹挑战者" |
| 达到第5层 | 200金币 |

---

### 2.3 第3周：极速挑战

**规则**：
- 每场战斗回合数上限：15回合
- 超过15回合强制平局，伤害加成重置
- 敌人生命-20%

**设计意图**：
- 鼓励速战速决
- 考验爆发能力

**限定奖励**：
| 达成条件 | 奖励 |
|----------|------|
| 通关 | 限定卡牌"闪电打击" |
| 达到第8层 | 头像"闪电侠" |
| 达到第5层 | 200金币 |

---

### 2.4 第4周：无限火力

**规则**：
- 所有技能冷却-1
- 初始怒气+30
- 敌人技能冷却+1

**设计意图**：
- 鼓励频繁使用技能
- 考验技能combo能力

**限定奖励**：
| 达成条件 | 奖励 |
|----------|------|
| 通关 | 限定卡牌"无限能量" |
| 达到第8层 | 头像"能量大师" |
| 达到第5层 | 200金币 |

---

## 3. 赛季任务

### 3.1 每周任务

## 4. 赛季排名系统

### 4.1 排名指标

| 指标 | 计分方式 |
|------|----------|
| 通关层数 | 每层100分 |
| 剩余生命 | 1% = 1分 |
| 剩余回合 | 1回合 = 10分 |
| 使用技能 | 1次 = 5分 |

### 4.2 排名奖励

| 排名 | 奖励 |
|------|------|
| #1 | 限定传奇卡牌 + 金色头像框 |
| #2-10 | 限定稀有卡牌 + 银色头像框 |
| #11-100 | 500金币 |
| 参与奖 | 200金币 |
