# 剧情文本设计

## 1. 剧情结构

### 1.1 章节划分

| 章节 | 内容 | 触发条件 |
|------|------|----------|
| 序章 | 角色背景 | 游戏开始 |
| 第1-4层 | 蒸汽劳工的故事 | 每层通关 |
| 第5层 | Boss战-葛罗普 | Boss战前/中/后 |
| 第6-9层 | 轮回教的秘密 | 每层通关 |
| 第10层 | Boss战-阿特拉斯 | Boss战前/中/后 |
| 尾声 | 通关结局 | 通关后 |

---

## 2. 角色对话

### 2.1 无名挑战者 序章

```
[画面：循环之塔底部，玩家站在入口前]

【旁白】
"你是被选中的虚像。"
"承载着无数人的希望，进入这座塔。"
"记住——"
"三拳定乾坤。"

【玩家】（内心独白）
"...我是谁？"
"为什么...我必须登上塔顶？"

[点击"进入塔内"]

【旁白】
"答案...在塔顶等你。"
```

---

### 2.2 第1层 - 蒸汽劳工

```
[遇到蒸汽劳工]

蒸汽劳工："劝你现在回头，塔里的路可不好走。"

【选项】
A. "我必须上去。"
B. "你是谁？"
```

**战后对话**：
```
蒸汽劳工："...有点本事。"
蒸汽劳工："但这才只是开始..."
蒸汽劳工："前面的守卫，可不会像我这么好对付。"
```

## 3. Boss战对话

### 3.1 第5层Boss - 葛罗普

**战前对话**：
```
[第5层入口]

葛罗普："欢迎来到第5层，挑战者。"
葛罗普："我是这里的执政官——葛罗普。"
葛罗普："齿轮...永动。"

【玩家】
"让开，我要去塔顶。"

葛罗普："塔顶？哈哈..."
葛罗普："那就先打破我的齿轮再说！"
```

**战中对话（血量<50%）**：
```
葛罗普："(惊讶) 竟然...伤到我..."
葛罗普："看来...不能小看你了。"
葛罗普："全功率运转！"
```

**战后对话**：
```
葛罗普："(单膝跪地) ...是我输了。"
葛罗普："你的实力，我认可了。"
葛罗普："但阿特拉斯大人...不会放过你的。"

【玩家】
"阿特拉斯？"

葛罗普："...你会知道的。"
葛罗普："祝你好运，挑战者。"
```

---

### 3.2 第8层 - 轮回教密信

**事件触发**：
```
[进入房间]

【玩家发现一封信件】

【信件内容】
"致阿特拉斯大人："
"新的虚像已经进入塔内。"
"根据预言，他将成为打破轮回的关键。"
"请务必阻止他。"

—— 轮回教 长老会

【选项】
A. "轮回教...他们在阻止我？"
B. "预言？那是什麼？"
C. [收好信件，继续前进]
```

### 3.3 第10层Boss - 阿特拉斯

**战前对话**：
```
[塔顶]

阿特拉斯："你终于来了，虚像。"
阿特拉斯："我等你...很久了。"

【玩家】
"你是谁？"

阿特拉斯："我是阿特拉斯，轮回的守护者。"
阿特拉斯："也是...这座塔的意志。"

【玩家】
"让我过去。"

阿特拉斯："过去？哈哈哈..."
阿特拉斯："你真的...了解自己在做什么吗？"
阿特拉斯："打破轮回...意味着所有的一切都会重置。"
阿特拉斯："你愿意...牺牲这一切吗？"

【选项】
A. "我别无选择。"
B. "我会找到其他办法。"
C. [直接战斗]
```

**战中对话（血量<50%）**：
```
阿特拉斯："(震惊) 竟然...伤得这么重..."
阿特拉斯："虚像的力量...比我想象的更强。"
阿特拉斯："但是！"
阿特拉斯："轮回...是不可打破的！"
```

**战后对话（胜利）**：
```
[阿特拉斯化为光芒消散]

阿特拉斯："(声音) 原来...这就是打破的感觉..."
阿特拉斯："我终于...理解了。"
阿特拉斯："去吧，虚像。"
阿特拉斯："创造...属于你的新轮回。"

【画面变白】

【旁白】
"轮回被打破了。"
"新的故事...由此开始。"
```

## 4. 结局文本

### 4.1 完美结局

```
[画面：塔顶的日出]

【玩家内心独白】
"我做到了..."
"打破轮回，创造新的可能。"

[俯瞰整座塔]

【玩家】
"这只是一个开始。"
"新的循环...将由我来守护。"

【系统提示】
"恭喜通关！"
"获得成就：轮回终结者"

[解锁隐藏结局]
```

## 5. 世界观文本

### 5.1 势力介绍

**齿轮会**：
> "秩序，是世界的根本。"
> "齿轮永动，万物有序。"
> "我们相信，三角循环是不可打破的法则。"

**轮回教**：
> "打破轮回，获得自由。"
> "命运，不应被循环束缚。"
> "我们是打破枷锁的钥匙。"

**蒸汽联盟**：
> "科技，创造未来。"
> "我们不参与纷争，只为世界进步而存在。"
> "蒸汽...是连接一切的纽带。"

### 5.2 背景设定

**循环素**：
> "存在于万物中的神秘能量。"
> "分为石头、剪刀、布三种形态。"
> "三角循环，维持着世界的平衡。"

**虚像**：
> "每个时代的变数。"
> "能够看到事物的本质——出拳倾向。"
> "是打破or维持循环的关键。"

## 6. 文本规格

### 6.1 对话格式

```
【角色名】
对话内容...

【选项】
A. 选项A
B. 选项B
C. 选项C
```

### 6.2 文本长度

| 类型 | 建议长度 |
|------|----------|
| 战前对话 | 50-100字 |
| 战中对话 | 30-50字 |
| 战后对话 | 30-80字 |
| 剧情文本 | 100-200字 |
| 旁白 | 20-50字 |

---

| 语言 | 状态 |
|------|------|
