## 3. 成就奖励机制

### 3.1 金币奖励

| 成就类型 | 平均奖励 | 总计 |
|----------|---------|------|
| 剧情成就 | 250 | 2500 |
| 战斗成就 | 180 | 2700 |
| 收集成就 | 300 | 3600 |
| 挑战成就 | 550 | 4400 |
| 隐藏成就 | 400 | 2000 |

**金币总计**：约15200金币

### 3.2 特殊奖励

| 奖励类型 | 来源 | 效果 |
|----------|------|------|
| 角色解锁 | 剧情成就 | 解锁新角色 |
| 专属卡牌 | 战斗成就 | 获得稀有/传奇卡 |
| 头像框 | 收集/隐藏 | 展示用装饰 |
| 称号 | 剧情/隐藏 | 玩家称号 |

## 5. 成就数据追踪

### 5.1 追踪变量

| 变量 | 用途 |
|------|------|
| totalDamage | 累计伤害 |
| totalWins | 累计胜场 |
| maxWinStreak | 最高连胜 |
| cardsCollected | 收集卡牌数 |
| playTime | 总游玩时间 |

### 5.2 统计界面

玩家可在统计页面查看：
- 总胜率
- 最高连胜
- 累计伤害
- 使用最多的卡牌
- 各角色通关次数
- 成就完成进度
