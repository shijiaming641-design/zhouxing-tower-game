# 核心机制

## 1. 猜拳基础系统

### 三元素关系
```
石头 → 剪刀 → 布 → 石头
(胜)  (胜)   (胜)
```

### 基础规则
| 结果 | 效果 |
|------|------|
| 胜利 | 造成全额伤害，+10怒气 |
| 失败 | 受到伤害，+5怒气 |
| 平局 | 双方各造成50%伤害，+5怒气 |

## 2. 策略深度

### 可见信息
- 敌人的出拳倾向（百分比显示）
- 敌人的怒气值
- 敌人的技能冷却状态

### 隐藏信息
- 玩家的出拳（同时 reveal）
- 部分敌人的特殊模式

## 3. 连胜/连击系统

| 连胜次数 | 伤害加成 | 特殊效果 |
|----------|----------|----------|
| 3连胜 | +20% | 获得1张临时牌 |
| 5连胜 | +50% | 恢复10%生命 |
| 7连胜 | +100% | 额外行动一次 |

## 4. 怒气机制

- **获取**: 每回合根据结果获得怒气
- **使用**: 怒气可释放强力技能
- **上限**: 100点
