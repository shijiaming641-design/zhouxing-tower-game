# 核心机制

## 1. 猜拳基础系统

### 三元素关系

石头 → 剪刀 → 布 → 石头
(胜) (胜) (胜)

### 基础规则

| 结果 | 效果 |
|------|------|
| 胜利 | 造成全额伤害，+10怒气 |
| 失败 | 受到伤害，+5怒气 |
| 平局 | 双方各造成50%伤害，+5怒气 → +7怒气（优化后） |

## 2. 策略深度

### 可见信息
- 敌人的出拳倾向（百分比显示）
- 敌人的怒气值
- 敌人的技能冷却状态

### 隐藏信息
- 玩家的出拳（同时 reveal）
- 部分敌人的特殊模式

## 3. 连胜/连击系统

| 连胜次数 | 伤害加成 | 特殊效果 |
|----------|----------|----------|
| 3连胜 | +20% | 获得1张临时牌 |
| 5连胜 | +50% | 恢复10%生命 |
| 7连胜 | +100% | 额外行动一次 |

## 4. 怒气机制

- 获取: 每回合根据结果获得怒气
- 使用: 怒气可释放强力技能
- 上限: 100点

## 5. 平局优化机制

### 问题：平局过多时的体验优化

| 机制 | 效果 |
|------|------|
| 平局连击 | 连续平局2次后，第3次平局触发"博弈升级"——本次出拳无法看到敌人倾向 |
| 平局惩罚 | 连续3次平局后，双方强制进入"对决模式"——石头/剪刀/布各出一人 |
| 平局补偿 | 平局时额外+2怒气（+5变+7），鼓励尽快打破僵局 |
| 虚张声势 | 平局时玩家可选择——假装变换出拳，实际不变，欺骗敌人消耗 |

## 6. 随机性与策略平衡

### 问题：随机性与策略的平衡点

**敌人倾向权重分配**：
- 60% 固定倾向（可被观察）
- 30% 可被卡牌/技能影响
- 10% 完全随机

**策略层级**：
| 层级 | 机制 | 策略深度 |
|------|------|----------|
| 信息层 | 敌人出拳倾向可见 | 基础预判 |
| 卡牌层 | 通过卡牌影响敌人倾向/修改出拳 | 中级策略 |
| 心理层 | 虚张声势、博弈升级、连胜心理战 | 高级博弈 |
| 风险层 | 高风险高回报路线选择 | 宏观策略 |

**核心原则**：每回合都有可决策点，避免"纯猜拳"
