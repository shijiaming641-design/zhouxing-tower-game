# 游戏概述

- **游戏名称**: 周行：循环三角
- **版本**: 0.1.0
- **文档日期**: 2026-03-12
- **游戏类型**: Roguelike / 策略 / 卡牌
- **核心玩法**: 猜拳策略爬塔

## 平台
- [x] PC (Steam)
- [x] iOS
- [x] Android
- [x] Web

## 核心体验
一句话描述：[玩家通过猜拳策略，层层攀登，击败塔顶Boss]

## 目标用户
- 年龄：12-35岁
- 偏好：策略游戏、Roguelike、轻度竞技
- 游戏时长：单局15-30分钟

## 核心循环
1. 选择路径 → 2. 遭遇战斗 → 3. 策略猜拳 → 4. 获取奖励 → 5. 强化角色 → 6. 挑战Boss

## 独特卖点
蒸汽朋克与赛璐璐的视觉碰撞，三拳定乾坤的读心博弈。没有运气，只有对敌人行为的预判与反制。Roguelike随机爬塔 + 猜拳策略深度 = 运筹帷幄的极限快感。
