## 独特卖点
蒸汽朋克与赛璐璐的视觉碰撞，三拳定乾坤的读心博弈。没有运气，只有对敌人行为的预判与反制。Roguelike随机爬塔 + 猜拳策略深度 = 运筹帷幄的极限快感。

## 世界观

### 背景故事
> 第三次工业革命后，煤炭与蒸汽取代了电力，成为维系世界的核心能源。
> 
> 然而，蒸汽的核心并非能源，而是一种名为**「循环素」**的神秘物质。
> 
> **「三角循环」**是这个世界的运转法则——万物皆在石头、剪刀、布的三重循环中彼此制约、动态平衡。
> 
> 传说塔顶存在一台终极蒸汽机，名为**「阿特拉斯」**，它维持着整个世界的循环素供给。数百年来，无数挑战者攀登「循环之塔」，试图掌控这台机器，成为新的世界秩序制定者。
> 
> 你，将成为下一个挑战者。

### 核心概念

| 概念 | 说明 |
|------|------|
| 循环素 | 蒸汽时代的核心能源，石头/剪刀/布三种形态 |
| 三角循环 | 世界运转的基本法则，万物相生相克 |
| 循环之塔 | 999层的巨型蒸汽机械塔，每层都是试炼 |
| 执政官 | 塔内各层的守护者，负责筛选挑战者 |
| 虚像 | 挑战者的意念实体化，可附身战斗 |

### 势力派系

| 派系 | 理念 | 代表角色 |
|------|------|----------|
| 齿轮会 | 秩序与规则，相信循环不可打破 | 葛罗普 |
| 轮回教 | 相信打破循环才能获得自由 | 阿特拉斯 |
| 蒸汽联盟 | 中立，致力维护世界运转 | 待定 |

### 角色设定

**主角**：无名挑战者
- 身份：被选中的"虚像"继承者
- 能力：可以看到敌人的"循环倾向"
- 目标：攀登至塔顶，改变世界秩序

**葛罗普**（Boss 1）
- 身份：第5层执政官，齿轮会核心成员
- 外形：半人半机械，齿轮构成的躯壳
- 性格：死板、秩序至上、不容打破规则

**阿特拉斯**（Boss 2）
- 身份：第10层守望者，轮回教首领
- 外形：三角形态的能量体，核心是一颗旋转的三角核心
- 性格：神秘、喜欢博弈、相信命运可被打破
