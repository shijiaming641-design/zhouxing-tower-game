# 战斗系统

## 1. 回合流程

```
┌─────────────────────────────────────────────────────────┐
│                    战斗回合流程                          │
├─────────────────────────────────────────────────────────┤
│  回合开始 → 观察敌人 → 选择出拳 → 同时揭示 → 结算伤害    │
│                                                    ↓    │
│  ←←←←←←←←←  检查技能触发  ←←←←←←←←←←  回合结束      │
└─────────────────────────────────────────────────────────┘
```

### 详细步骤

| 步骤 | 玩家操作 | 系统处理 |
|------|----------|----------|
| 1. 回合开始 | - | 重置临时buff、检查被动技能 |
| 2. 观察敌人 | 查看敌人倾向/怒气/状态 | 显示敌人信息面板 |
| 3. 选择出拳 | 选择石头/剪刀/布 + 使用技能 | 锁定选择 |
| 4. 同时揭示 | - | 双方出拳动画同时展示 |
| 5. 结算伤害 | - | 计算伤害、处理生死 |
| 6. 检查技能 | - | 触发被动/反应技能 |
| 7. 回合结束 | - | 更新怒气/状态持续回合 |

### 战斗限时
- 每回合选择时限：**30秒**
- 超时默认：随机出拳
- 倒计时显示：UI右上角

## 2. 伤害计算

### 基础公式

```
实际伤害 = (攻击力 - 防御力) × 克制系数 × 连胜加成 × 随机波动(0.9-1.1)
```

### 克制系数表

| 玩家出拳 | 敌人出拳 | 结果 | 克制系数 |
|----------|----------|------|----------|
| 石头 | 剪刀 | 胜利 | 1.5x |
| 剪刀 | 布 | 胜利 | 1.5x |
| 布 | 石头 | 胜利 | 1.5x |
| 石头 | 石头 | 平局 | 0.75x |
| 剪刀 | 剪刀 | 平局 | 0.75x |
| 布 | 布 | 平局 | 0.75x |
| 石头 | 布 | 失败 | 0.5x |
| 剪刀 | 石头 | 失败 | 0.5x |
| 布 | 剪刀 | 失败 | 0.5x |

### 连胜加成表

| 连胜次数 | 伤害加成 | 特殊效果 |
|----------|----------|----------|
| 1-2胜 | 1.0x | 无 |
| 3胜 | 1.2x | 获得1张临时牌 |
| 5胜 | 1.5x | 恢复10%生命 |
| 7胜 | 2.0x | 额外行动1次 |

### 伤害示例

| 场景 | 计算 | 结果 |
|------|------|------|
| 初始玩家攻击普通怪胜利 | (10-2)×1.5×1.0 | 12 |
| 初始玩家攻击普通怪胜利+3连胜 | (10-2)×1.5×1.2 | 14.4 |
| 初始玩家攻击普通怪失败 | (10-2)×0.5×1.0 | 4 |
| 初始玩家攻击普通怪平局 | (10-2)×0.75×1.0 | 6 |

## 3. 技能系统

### 技能类型

| 类型 | 触发方式 | 示例 |
|------|----------|------|
| 被动技能 | 始终生效 | 预判、齿轮护盾 |
| 主动技能 | 消耗怒气释放 | 重拳、命运枷锁 |
| 反应技能 | 特定条件触发 | 反击、虚张声势 |

### 玩家技能库

| 技能名 | 类型 | 消耗 | 冷却 | 效果 |
|--------|------|------|------|------|
| 重拳 | 主动 | 30怒 | 2回合 | 伤害+100%，无视防御 |
| 预判 | 被动 | - | - | 敌人出拳倾向显示+20%精度 |
| 反击 | 反应 | 20怒 | 1回合 | 被克制时触发，反弹50%伤害 |
| 防守 | 主动 | 15怒 | 1回合 | 获得8点护盾 |
| 集怒 | 主动 | - | 2回合 | 额外获得10点怒气 |
| 虚张声势 | 反应 | 10怒 | 3回合 | 平局时，敌人下回合倾向反转 |

### 敌人技能库

| 技能名 | 所属 | 类型 | 效果 |
|--------|------|------|------|
| 重锤 | 葛罗普 | 主动 | 伤害+80%，无视防御 |
| 齿轮护盾 | 葛罗普 | 被动 | 每5回合获得1层护盾 |
| 过载 | 葛罗普(Boss) | 阶段3 | 每回合自动释放重锤 |
| 命运枷锁 | 阿特拉斯 | 主动 | 下回合玩家必出被克制拳 |
| 时间回溯 | 阿特拉斯 | 被动 | 每损失25%血量重置技能冷却 |
| 无限轮回 | 阿特拉斯(Boss) | 阶段3 | 平局时恢复10%生命 |

### 技能冷却机制

- 主动技能：有冷却回合数
- 被动技能：持续生效
- 反应技能：触发后进入冷却
- Boss技能：部分特殊机制

## 4. AI 设计

### 难度层级

| 层级 | 敌人类型 | 特点 | 可被学习 |
|------|----------|------|----------|
| 简单 | 蒸汽劳工 | 完全随机，无明显倾向 | 否 |
| 普通 | 齿轮守卫 | 有固定倾向，可被学习 | 是 |
| 困难 | 蒸汽监工 | 会学习玩家模式，偶尔反制 | 是 |
| Boss | 葛罗普/阿特拉斯 | 多阶段，阶段转换有策略变化 | 是 |

### AI决策逻辑

```
AI决策流程:
│
 ├─ 读取玩家历史出拳 (最近5回合)
 ├─ 检查当前阶段行为模式
 ├─ 权重计算: 60%固定倾向 + 30%可变量 + 10%随机
 └─ 输出出拳选择
```

### 敌人行为模式

| 敌人 | 模式 | 出拳倾向 | 特殊行为 |
|------|------|----------|----------|
| 蒸汽劳工 | 随机 | 33/33/34 | 无 |
| 齿轮守卫 | 偏石 | 60/20/20 | 防御姿态 |
| 管线鼠 | 敏捷 | 30/50/20 | 高攻低血 |
| 蒸汽监工 | 攻击 | 20/30/50 | 攻击附带易伤 |
| 齿轮骑士 | 坚守 | 70/15/15 | 高防御 |
| 葛罗普 | 三阶段 | 详见04章 | 阶段转换 |
| 阿特拉斯 | 博弈 | 详见04章 | 预判+限制出拳 |

## 5. 状态效果

### 状态类型

| 状态 | 效果 | 持续回合 | 可叠加 | 移除方式 |
|------|------|----------|--------|----------|
| 脆弱 | 受到伤害+50% | 2 | 是 | 回合结束 |

```

```

## 6. 特殊战斗机制

### 6.1 平局对决模式

**触发条件**：连续3次平局

**机制**：
- 双方强制进入"对决模式"
- 连续进行3轮快速出拳（每轮10秒）
- 每轮出拳后立即揭示并结算
- 3轮结束后回归正常战斗

### 6.2 博弈升级

**触发条件**：连续2次平局

**机制**：
- 第3次平局时，玩家无法看到敌人出拳倾向
- 增加心理博弈深度

### 6.3 连胜追击

**触发条件**：7连胜

**机制**：
- 获得额外1次行动机会
- 可连续出拳2次
- 两次出拳分别结算

### 6.4 绝境反击

**触发条件**：玩家HP < 20%

**机制**：
- 攻击力临时+30%
- 暴击率+20%（克制伤害变为2.0x）
- 效果持续至本场战斗结束

## 7. 战斗结算

### 胜利结算

```
├─ 获得金币: 10-20 (普通) / 30-50 (精英) / 100 (Boss)
├─ 获得卡牌: 60%普 / 30%稀 / 10%传
├─ 通关奖励: 200金币 (通关时)
└─ 继续/升级/商店
```

### 失败结算

```
├─ 生命值: 降至0时失败
├─ 评价系统: S/A/B/C/D
│   ├─ S: 无伤/低伤通关
│   ├─ A: 损失<30%生命
│   ├─ B: 损失<60%生命
│   ├─ C: 损失<90%生命
│   └─ D: 失败
├─ 统计数据: 伤害/回合数/技能使用
└─ 重新开始/退出
```

### 回合中断条件

| 条件 | 处理 |
|------|------|
| 敌人HP ≤ 0 | 战斗胜利 |
| 玩家HP ≤ 0 | 战斗失败 |
| 玩家投降 | 战斗失败 |
| 30回合上限 | 强制平局，伤害加成重置 |

## 8. 战斗UI交互

### 战斗界面元素

| 区域 | 内容 |
|------|------|
| 顶部 | 回合数、战斗计时器 |
| 左侧 | 敌人信息(HP/怒气/状态/倾向) |
| 中间 | 战斗动画区域 |
| 右侧 | 玩家信息(HP/怒气/状态) |
| 底部 | 出拳按钮(石头/剪刀/布)、技能栏 |
| 手牌区 | 当前持有卡牌 |

### 出拳交互流程

```
1. 点击 [石头/剪刀/布] 按钮
   ↓
2. 按钮高亮确认，可更换
   ↓
3. (可选) 点击技能按钮消耗怒气
   ↓
4. 倒计时结束 / 点击"确认"
   ↓
5. 双方出拳动画同时播放
   ↓
6. 伤害数字弹出，状态更新
```
