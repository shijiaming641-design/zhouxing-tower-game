# 战斗系统

## 1. 回合流程

```
回合开始 → 观察敌人 → 选择出拳 → 同时揭示 → 结算伤害 → 检查技能 → 回合结束
```

## 2. 伤害计算

### 基础公式
```
实际伤害 = (攻击力 - 防御力) × 克制系数 × 连胜加成 × 随机波动(0.9-1.1)
```

### 克制系数
| 克制关系 | 系数 |
|----------|------|
| 正常克制 | 1.5x |
| 被克制 | 0.5x |
| 平局 | 0.75x |

## 3. 技能系统

### 技能类型
1. **被动技能**: 始终生效
2. **主动技能**: 消耗怒气释放
3. **反应技能**: 特定条件触发

### 示例技能表
| 技能名 | 类型 | 消耗 | 效果 |
|--------|------|------|------|
| 重拳 | 主动 | 30怒 | 伤害+100%，无视防御 |
| 预判 | 被动 | - | 敌人出拳倾向显示+20%精度 |
| 反击 | 反应 | 20怒 | 被克制时触发，反弹50%伤害 |

## 4. AI 设计

### 难度层级
| 层级 | 特点 |
|------|------|
| 简单 | 完全随机，无明显倾向 |
| 普通 | 有固定倾向，可被学习 |
| 困难 | 会学习玩家模式，偶尔反制 |
| Boss | 多阶段，阶段转换有策略变化 |

## 5. 状态效果

| 状态 | 效果 | 持续 |
|------|------|------|
| 脆弱 | 受到伤害+50% | 2回合 |
| 护盾 | 免疫一次伤害 | 1回合 |
| 狂暴 | 伤害+30%，但更易被预判 | 3回合 |
