# 爬塔关卡设计

## 1. 塔结构

```
第10层 ──── Boss战（最终Boss）
   │
 9-6层 ──── 精英关卡 + 事件
   │
 5-2层 ──── 普通战斗 + 商店 + 休息
   │
 第1层 ──── 教学关 + 起始选择
```

## 2. 节点类型

| 图标 | 类型 | 说明 |
|------|------|------|
| ⚔️ | 普通战斗 | 标准敌人，击败获得金币/卡牌 |
| 👹 | 精英战斗 | 强力敌人，奖励更丰厚 |
| 🏪 | 商店 | 购买卡牌、道具、恢复 |
| ❓ | 随机事件 | 可能获得/失去资源 |
| 🏕️ | 休息点 | 恢复生命或升级卡牌 |
| 👑 | Boss战 | 每5层一次，决定能否继续 |

## 3. 路线选择

每关结束后，玩家选择下一节点：
- 高风险路线：更多战斗，更好奖励
- 安全路线：更多休息/商店
- 平衡路线：混合节点
