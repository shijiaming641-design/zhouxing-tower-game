# 局外成长系统设计

> 版本：1.0  
> 日期：2026-03-14  
> 状态：已完成

---

## 1. 概述

局外成长系统是玩家在多周目中累积强化的机制，通过"人员存储器"解锁新能力，让后续游戏越来越强。

---

## 2. 核心资源：人员存储器

### 2.1 获得方式

| 来源 | 数量 | 说明 |
|------|------|------|
| 击败BOSS | 1-3 | 每层BOSS不同 |
| 成就奖励 | 1-5 | 一次性 |
| 挑战模式 | 1-10 | 额外挑战 |

### 2.2 存储上限

- 最大持有：99个
- 用于解锁和升级

---

## 3. 解锁项目

### 3.1 解锁树

```
人员存储器消耗
     │
     ├── 基础连招槽位 (1个)
     │      └── 1个槽 → 2个槽
     │
     ├── 基础属性提升 (1-2个)
     │      ├── +2最大生命
     │      └── +1攻击力
     │
     ├── 背包扩展 (1个)
     │      └── +4格子
     │
     ├── 高级连招 (2个)
     │      └── 解锁更强连招
     │
     ├── 连招冷却缩减 (2个)
     │      └── 所有连招-1回合
     │
     ├── 装备槽位 (2个)
     │      └── +1装备栏
     │
     ├── 终极连招 (4个)
     │      └── 解锁终极技能
     │
     ├── 永久被动 (4个)
     │      └── 每局开始+1护甲
     │
     ├── 外挂充能 (4个)
     │      └── 每局可用次数+1
     │
     ├── 保护光环 (8个)
     │      └── 免疫一次伤害
     │
     └── 属性强化 (8个)
            └── 血量+2，攻击+0.5
```

### 3.2 解锁优先级建议

| 优先级 | 项目 | 性价比 |
|--------|------|--------|
| ⭐⭐⭐ | 基础连招槽位 | 核心机制 |
| ⭐⭐⭐ | 背包扩展 | 提升体验 |
| ⭐⭐ | 基础属性 | 前期优势 |
| ⭐⭐ | 高级连招 | 战力提升 |
| ⭐ | 装备槽位 | 中期强化 |

## 4. 多周目机制

### 4.1 周目等级

| 周目 | 解锁内容 | 敌人强度 |
|------|----------|----------|
| 1 | 基础游戏 | 1.0x |
| 2 | 高级卡牌 | 1.2x |
| 3 | 终极技能 | 1.5x |
| 4+ | 无限模式 | 2.0x+ |

### 4.2 继承内容

| 类型 | 继承 | 说明 |
|------|------|------|
| 人员存储器 | ✅ 100% | 保留所有 |
| 解锁内容 | ✅ 100% | 永久解锁 |
| 装备 | ❌ | 不继承 |
| 金币 | ❌ | 不继承 |
| 卡牌 | ✅ 已解锁 | 可用 |

### 4.3 周目奖励

| 周目 | 额外奖励 |
|------|----------|
| 通关第1周目 | +3 人员存储器 |
| 通关第2周目 | +5 人员存储器 |
| 通关第3周目 | +10 人员存储器 |
| 每后续周目 | +10 |

## 6. 代码结构

### 6.1 核心类

```csharp
// 玩家进度数据
[System.Serializable]
public class PlayerProgress
{
    public int memoryFragments;      // 人员存储器数量
    public List<string> unlockedCombos;  // 已解锁连招
    public List<int> unlockedLevels; // 已解锁内容
    public int currentWeek;          // 当前周目
    public int totalWins;             // 总胜场
}

// 局外成长管理器
public class ProgressionSystem
{
    private PlayerProgress progress;
    
    public bool Unlock(string itemId);
    public int GetMemoryCount();
    public void AddMemory(int count);
    public bool CanAfford(int cost);
    public void SaveProgress();
    public void LoadProgress();
}
```

### 6.2 文件位置

```
Core/
└── ProgressionSystem.cs    # 局外成长系统
```
