# 商店系统设计

> 版本：1.0  
> 日期：2026-03-14  
> 状态：已完成

---

## 1. 概述

商店系统为玩家提供购买装备、道具、卡牌的场所，是游戏内经济循环的重要环节。

---

## 2. 商店类型

### 2.1 商店枚举

```csharp
public enum ShopType
{
    Equipment,    // 装备商店
    Card,         // 卡牌商店
    Item,         // 道具商店
    Mystery,      // 神秘商人
    BlackMarket   // 黑市
}
```

### 2.2 商店特点

| 类型 | 刷新 | 折扣 | 特殊商品 |
|------|------|------|----------|
| 装备商店 | 每层 | 无 | 稀有装备 |
| 卡牌商店 | 每层 | 无 | 高级卡牌 |
| 道具商店 | 每次访问 | 无 | 消耗品 |
| 神秘商人 | 随机 | 9折 | 随机商品 |
| 黑市 | 随机 | 1.5倍 | 禁用商品 |

## 3. 商品设计

| 稀有度 | 概率 | 价格范围 | 属性数量 |

### 3.3 道具商品

## 4. 价格体系

### 4.1 基础公式

```
售价 = 基础价格 × 稀有度系数 × 状态系数
```

### 4.2 系数表

| 稀有度 | 稀有度系数 |
|--------|------------|
| 普通 | 1.0 |
| 精良 | 1.5 |
| 稀有 | 2.5 |
| 传说 | 5.0 |

### 4.3 玩家出售

- 玩家出售价格为购入价的 30%
- 稀有物品可能有特殊价值

---

## 5. 刷新机制

### 5.1 自动刷新

- 每进入新层刷新一次
- 休息点后刷新一次

### 5.2 手动刷新

```csharp
// 刷新费用递增
int refreshCost = baseCost * (refreshCount + 1);
// 首次免费
```

### 5.3 商品数量

| 商店类型 | 商品数量 | 可购买上限 |
|----------|----------|------------|
| 装备商店 | 4-6件 | 每件1次 |
| 卡牌商店 | 3-5张 | 每张1次 |
| 道具商店 | 6-8件 | 库存制 |

---

## 6. 交互流程

```
1. 进入商店
   ↓
2. 显示商品列表
   ↓
3. 玩家选择商品
   ├─ 购买 → 扣除金币 → 获得商品
   ├─ 出售 → 添加商品 → 获得金币
   └─ 刷新 → 消耗金币 → 新商品
   ↓
4. 离开商店
```

## 7. 代码结构

### 7.1 核心类

```csharp
// 商品基类
public class ShopItem
{
    public string ID;
    public string Name;
    public int Price;
    public int Stock;  // -1表示无限
}

// 商店管理器
public class ShopSystem
{
    private ShopType currentShopType;
    private List<ShopItem> items;
    private int refreshCount;
    
    public void OpenShop(ShopType type);
    public bool BuyItem(string itemId);
    public bool SellItem(string itemId);
    public void Refresh();
    public List<ShopItem> GetItems();
}
```
