# UI/UX 设计

## 1. 界面流程

```
主菜单 → 角色选择 → 塔层地图 → 战斗界面 → 结算 → 继续/结束
```

## 2. 战斗界面布局

```

```

## 4. 交互原则

1. **即时反馈**: 出拳后立即显示结果
2. **清晰可读**: 伤害数字大且明显
3. **降低焦虑**: 显示敌人倾向而非完全随机
4. **进度感**: 塔层进度清晰可见
