# UI/UX 设计

## 1. 界面流程

```
主菜单 → 角色选择 → 塔层地图 → 战斗界面 → 结算 → 继续/结束
```

## 2. 战斗界面布局

```
┌─────────────────────────────┐
│  敌人信息 (HP/怒气/Buffs)    │
│                             │
│      [敌人角色形象]          │
│                             │
│    出拳倾向: 石40%剪30%布30% │
├─────────────────────────────┤
│        VS 倒计时            │
├─────────────────────────────┤
│                             │
│      [玩家角色形象]          │
│                             │
│  [石头] [剪刀] [布]         │
│                             │
│  技能按钮 (怒气值显示)       │
├─────────────────────────────┤
│  玩家信息 (HP/怒气/手牌)     │
└─────────────────────────────┘
```

## 3. 视觉风格

### 主题
蒸汽朋克 × 赛璐璐 —— 齿轮与蒸汽的机械美学，搭配明快锐利的赛璐璐上色，形成独特的"复古未来"视觉冲击。UI强调金属质感与锐利线条的碰撞。

### 主色调
| 元素 | 颜色 | 色值 |
|------|------|------|
| 主色（蒸汽） | 黄铜金 | #B8860B |
| 辅色（机械） | 钢铁灰 | #4A4A4A |
| 点缀（能量） | 霓虹蓝 | #00D4FF |
| 背景 | 深铜褐 | #2C2416 |
| 胜利 | 翡翠绿 | #50C878 |
| 失败 | 胭脂红 | #E30022 |

### 字体
- 标题：Bebas Neue / 站酷高端黑
- 正文：Source Han Sans / 思源黑体
- 数字/数据：Orbitron

### 赛璐璐风格特点
- 高对比黑色轮廓线，色块分明
- 少渐变，多平涂
- 金属高光带橙色反光

## 4. 即时反馈
- 出拳：蒸汽喷射特效
- 连胜：齿轮加速动画
- 伤害数字：金属弹出效果

## 5. 交互原则
1. **即时反馈**: 出拳后立即显示结果
2. **清晰可读**: 伤害数字大且明显
3. **降低焦虑**: 显示敌人倾向而非完全随机
4. **进度感**: 塔层进度清晰可见
