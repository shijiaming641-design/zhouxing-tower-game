# 音效与音乐设计

## 1. 概述

游戏采用**蒸汽朋克+赛璐璐**风格，音效和音乐也相应融合了机械质感与明快节奏。

---

|------|------|

| 项目 | 内容 |

## 4. 音效参数

### 4.1 音量标准

| 类型 | 基准音量 | 备注 |
|------|----------|------|
| BGM | -12dB | 背景层 |
| 战斗音效 | -6dB | 强调层 |
| UI音效 | -3dB | 反馈层 |
| 语音 | -6dB | 清晰可懂 |

### 4.2 空间音效

| 场景 | 效果 |
|------|------|
| 战斗 | 适度混响，回声感 |
| 菜单 | 干声为主 |
| 室内（商店） | 小空间混响 |
| 室外（塔顶） | 开阔感 |

## 6. 语音设计

### 6.1 角色语音

| 角色 | 语音风格 |
|------|----------|
| 无名挑战者 | 沉稳少言，只有战斗时的简短喊声 |
| 齿轮少女 | 活泼自信，战斗中会有机械相关台词 |
| 轮回刺客 | 冷酷简短，战斗中会有嘲讽台词 |

### 6.2 战斗语音示例

**齿轮少女 萝丝 - 战斗**：
- "尝尝这个！"（使用技能）
- "齿轮可是不会停下的！"
- "3...2...1...哈哈！"

**轮回刺客 夜刃 - 战斗**：
- "太慢了。"
- "游戏结束。"
- "下一击，解决你。"

### 6.3 Boss语音

**葛罗普**：
- "秩序是不可打破的！"
- "齿轮...永动！"
- "你只是在拖延必然的失败。"

**阿特拉斯**：
- "欢迎来到...轮回的尽头。"
- "预判我的行动？可笑。"
- "这就是...命运。"

## 7. 音频技术规格

| 项目 | 规格 |
|------|------|
| 采样率 | 48kHz / 24bit |
| 输出格式 | MP3 320kbps / OGG |
| 动态范围 | DR12-14 |
| 平台 | Windows / macOS / iOS / Android |

### 音频管理器功能
- BGM淡入淡出（1秒）
- 音效优先级分组
- 音量预设（静音/低/中/高）
- 语音/音效独立音量
